# Simple Sidebar Website

This is a **simple static website** built using **HTML and CSS**. The project demonstrates a basic webpage layout with a **vertical sidebar navigation menu** and a **main content section**.  

The sidebar contains navigation links and a logo, while the main section displays a welcome message, descriptive text, and images. The layout is implemented using **CSS Flexbox** to create a responsive horizontal structure where the sidebar stays on the left and the content appears on the right.

This project is useful for learning **basic web layout design, Flexbox positioning, and simple responsive styling**.

## Installation

Step-by-step instructions on how to run the project locally.

1. **Clone the Repository:**
   - Clone the project repository from GitHub:
     ```bash
     git clone https://github.com/yourusername/simple-sidebar-website.git
     ```
   - Or download the repository as a ZIP file and extract it.

2. **Open the Project Folder:**
   - Open the folder using **Visual Studio Code**, **Sublime Text**, or any code editor.

3. **Run the Website:**
   - Open the `index.html` file in your browser.
   - You can also right-click the file and select **Open with Live Server** if using VS Code.

## Usage

Once the website is opened in a browser:

- The **left sidebar** contains:
  - Logo
  - Navigation links (Home, Team, About, Contact)

- The **main content area** displays:
  - A welcome heading
  - A paragraph of placeholder text
  - Two images displayed using Flexbox layout

The layout uses **CSS Flexbox** to keep the sidebar fixed on the left while the main content expands on the right.

## Project Structure

- **index.html** – Main webpage structure  
- **style.css** – Styling and layout using CSS Flexbox  
- **images/** – Contains images displayed on the webpage
